# Blindfold Chess

可互动的盲棋项目。

## 项目结构

```
blindfold-chess/
├── index.html      # 主入口（正在开发的页面）
├── .gitignore      # Git 忽略规则
├── README.md       # 项目说明（本文件）
├── AGENTS.md       # AI 规则手册（触发词、恢复流程）
├── STATE.md        # 动态状态备份（开发进度、待办）
└── 备忘录.md        # 命令速查表、技术概念解释
```

## 技术决策

| 决策项 | 选择 | 原因 |
|--------|------|------|
| 前端技术栈 | **HTML + CSS + JS** | 项目规模小，纯手写完全够用，无需引入框架 |
| 框架评估 | **暂不使用 React / Vue** | 当前只需一个可互动的棋盘页面，框架会增加不必要的复杂度 |
| 后续扩展 | 如需增加在线对战、用户系统等功能，再评估引入框架 |
