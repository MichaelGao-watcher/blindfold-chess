# Task: BlindfoldModule（盲棋对战模块）

> 模块状态：已实现（需完善）  
> 目标：与 Stockfish 进行盲棋对弈，支持难度选择、走子输入、终局判定、PGN 导出  
> 设计文档：high-Level Design.md §4.2

---

## 子任务清单

### 难度选择
- [ ] **B-01** 创建难度选择界面（Easy / Medium / Hard / Expert）
- [ ] **B-02** 难度按钮样式：当前选中高亮、悬停反馈
- [ ] **B-03** 选择难度后调用 `BlindfoldModule.init(level)` 初始化新对局

### 对局界面
- [ ] **B-04** 创建盲棋对战界面 HTML 结构：走子输入区 + 棋盘区 + 走法列表区 + 控制按钮
- [ ] **B-05** 棋盘显示/隐藏切换按钮与逻辑（`BlindfoldModule.toggleBoard()`）
- [ ] **B-06** 棋盘默认隐藏（盲棋模式），用户可手动显示
- [ ] **B-07** 执黑时对局：棋盘反转 180°，坐标标注随之反转（黑方视角）

### 走子输入与验证
- [ ] **B-08** 走子输入框（支持代数记谱法：e4, Nf3, O-O 等）
- [ ] **B-09** 输入框回车提交走子
- [ ] **B-10** 代数记谱法验证：调用 chess.js 检查走法是否合法
- [ ] **B-11** 非法走法提示：输入框震动 + 红色提示文字
- [ ] **B-12** 合法走法确认：播放走子音效 → 更新内部局面 → 触发引擎思考

### 引擎交互
- [ ] **B-13** 用户走子后调用 `EngineModule.setPosition(fen)` 设置局面
- [ ] **B-14** 调用 `EngineModule.go(callback)` 触发引擎思考
- [ ] **B-15** 接收引擎 bestmove，解析并更新内部局面
- [ ] **B-16** 引擎走子后播放走子音效，更新走法列表
- [ ] **B-17** 引擎思考中显示加载状态（可选："引擎思考中..."）

### 引擎候选走法（辅助功能）
- [ ] **B-18** 集成 `EngineModule.goMultiPv()` 获取 Top 3 候选走法 + 引擎评分
- [ ] **B-19** 在走子输入框上方展示候选走法列表（如：`1. e4 (+0.5)  2. d4 (+0.3)  3. Nf3 (+0.1)`）
- [ ] **B-20** 候选走法辅助受设置开关控制（默认关闭，由 SettingsModule 控制）

### 走法历史与 PGN
- [ ] **B-21** 走法列表实时更新（带序号：1. e4 e5  2. Nf3 Nc6）
- [ ] **B-22** 走法列表支持滚动查看
- [ ] **B-23** PGN 格式生成（包含事件标签、走法序列、结果）
- [ ] **B-24** 对局结束后显示结果卡片，卡片上展示完整 PGN 走法列表
- [ ] **B-25** PGN 复制按钮（一键复制到剪贴板）

### 终局判定
- [ ] **B-26** 将杀判定：检测引擎或用户是否被将杀
- [ ] **B-27** 逼和判定：检测是否无合法走子且未被将军
- [ ] **B-28** 和棋判定：三次重复局面、50 步规则、双方同意和棋（如需要）
- [ ] **B-29** 认输功能：`BlindfoldModule.resign()` 按钮
- [ ] **B-30** 终局后触发 `onGameOver(callback)`，通知 StatsModule 和 StorageModule

### 自动保存
- [ ] **B-31** 对局结束时自动调用 `StorageModule.addGameRecord(record)` 保存记录
- [ ] **B-32** 对局记录数据结构：`{ id, date, mode, difficulty, result, pgn, moves, duration }`

### 控制按钮
- [ ] **B-33** "再来一局"按钮：调用 `BlindfoldModule.newGame()`
- [ ] **B-34** "返回主菜单"按钮：导航回欢迎页
- [ ] **B-35** 对局进行中显示当前难度、当前回合（白/黑）

### 测试与优化
- [ ] **B-36** 测试各难度下引擎是否正常回应
- [ ] **B-37** 测试非法走法是否被拒绝
- [ ] **B-38** 测试终局判定（将杀/逼和/和棋）
- [ ] **B-39** 测试 PGN 格式是否符合标准
- [ ] **B-40** 测试自动保存是否在对局结束时触发
