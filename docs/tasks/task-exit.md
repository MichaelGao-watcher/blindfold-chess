# Task: ExitModule（退出模块）

> 模块状态：规划中  
> 目标：处理退出流程，随机展示幽默台词  
> 设计文档：high-Level Design.md §4.8

---

## 子任务清单

### 退出界面
- [ ] **E-01** 创建退出界面 HTML 结构：台词展示区 + 确认/取消按钮
- [ ] **E-02** 调用 `ExitModule.init()` 初始化退出界面
- [ ] **E-03** `ExitModule.showExitScreen()` 显示退出屏幕

### 幽默台词库
- [ ] **E-04** 台词数据结构：`{ zh: string, en: string, style: string }`
- [ ] **E-05** 中文台词 1："这对我来说太难了，我要去打开 TikTok 放松一下"
- [ ] **E-06** 中文台词 2："一代天才，就此陨落"
- [ ] **E-07** 英文台词 1："This is too hard for me. I'm going to open TikTok and relax."
- [ ] **E-08** 英文台词 2："A brilliant mind, fallen just like that."
- [ ] **E-09** 扩展更多台词（至少 10 条，混合幽默风格）

### 台词展示逻辑
- [ ] **E-10** `ExitModule.getRandomLine()` 随机获取一条台词
- [ ] **E-11** 随机算法确保不连续重复同一条
- [ ] **E-12** 根据当前语言设置显示对应语言版本
- [ ] **E-13** 台词展示动画：打字机效果或淡入效果

### 退出流程
- [ ] **E-14** 触发退出时机：用户点击浏览器关闭/刷新，或点击"退出"按钮
- [ ] **E-15** 页面内退出按钮：显示退出界面 → 用户确认后返回欢迎页或关闭标签
- [ ] **E-16** 取消退出：用户点击"取消"或"再玩一局"返回原界面
- [ ] **E-17** 退出前自动保存当前进行中的对局状态（如有）

### 视觉设计
- [ ] **E-18** 退出界面风格：与整体苹果风格一致，居中卡片式布局
- [ ] **E-19** 台词字体：稍大、醒目，带有情感色彩
- [ ] **E-20** 按钮设计："确认退出"（次要样式）+ "再玩一局"（主要样式）

### 测试与优化
- [ ] **E-21** 测试台词随机性覆盖（多次触发不重复率）
- [ ] **E-22** 测试中英切换时台词是否正确
- [ ] **E-23** 测试取消退出后是否正常返回原界面
