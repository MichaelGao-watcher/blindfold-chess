# Task: CoordinateModule（坐标练习模块）

> 模块状态：已实现（需完善）  
> 目标：通过 A/B 两种模式练习棋盘坐标，支持计时挑战和视角切换  
> 设计文档：high-Level Design.md §4.3

---

## 子任务清单

### 练习设置界面
- [ ] **C-01** 创建坐标练习设置界面 HTML：模式选择 + 执白/执黑 + 计时设置
- [ ] **C-02** 模式 A（找格子）与模式 B（报坐标）切换按钮
- [ ] **C-03** 执白/执黑视角切换按钮（影响棋盘方向）
- [ ] **C-04** 计时设置：30s / 60s / 120s / 不限时（0s）选项
- [ ] **C-05** "开始练习"按钮：调用 `CoordinateModule.startPractice(side, mode)`

### 模式 A：找格子
- [ ] **C-06** 系统随机报一个坐标（语音或文字显示）
- [ ] **C-07** 用户点击棋盘上的对应格子
- [ ] **C-08** 点击后判定正确/错误：正确 → 下一题；错误 → 震动反馈 + 高亮正确答案
- [ ] **C-09** 错误后强制用户点击正确格子才能继续
- [ ] **C-10** 确保 64 个坐标在练习中随机覆盖（不重复直到全部完成）

### 模式 B：报坐标
- [ ] **C-11** 系统随机高亮棋盘上的一个格子
- [ ] **C-12** 用户通过输入框输入坐标（如 e4）
- [ ] **C-13** 输入后判定正确/错误：正确 → 下一题；错误 → 输入框震动 + 显示正确答案
- [ ] **C-14** 错误后强制用户输入正确坐标才能继续

### 计时挑战
- [ ] **C-15** 倒计时显示（大字体，醒目位置）
- [ ] **C-16** 计时器到 0 时自动停止练习，显示结果
- [ ] **C-17** `CoordinateModule.setTimer(seconds)` 支持自定义时长
- [ ] **C-18** 不限时模式：隐藏计时器，用户自行结束

### 得分与反馈
- [ ] **C-19** 实时得分统计：正确数 / 总题数 / 正确率
- [ ] **C-20** 答对时：格子绿色高亮 + 得分 +1
- [ ] **C-21** 答错时：格子红色高亮 + 震动动画 + 正确答案闪烁提示
- [ ] **C-22** 连击计数（可选）：连续答对显示连击数

### 执白/执黑视角
- [ ] **C-23** 执白：标准视角（a1 在左下，h8 在右上）
- [ ] **C-24** 执黑：棋盘反转 180°，坐标标注也随之反转（a1 在右上，h8 在左下）
- [ ] **C-25** 视角切换后 `BoardRenderer` 正确重新渲染

### 练习结果
- [ ] **C-26** 练习结束界面：显示总题数、正确数、正确率、用时
- [ ] **C-27** 结果保存到 StorageModule（坐标练习成绩记录）
- [ ] **C-28** "再来一次"按钮：重置练习
- [ ] **C-29** "返回设置"按钮：回到练习设置界面

### 数据统计
- [ ] **C-30** 坐标练习历史记录查询接口
- [ ] **C-31** 根据正确率和速度评定等级（可选）

### 测试与优化
- [ ] **C-32** 测试 64 个坐标随机覆盖无遗漏
- [ ] **C-33** 测试答错后必须点击正确格子才能继续
- [ ] **C-34** 测试执黑视角下坐标标注是否正确反转
- [ ] **C-35** 测试计时器到点后是否正确停止并显示结果
- [ ] **C-36** 测试时长设置是否生效
