# Task: BoardRenderer（棋盘渲染模块）

> 模块状态：已实现（需完善）  
> 目标：通用棋盘渲染，支持棋子、坐标、视角反转、高亮、动画  
> 设计文档：high-Level Design.md §4.9

---

## 子任务清单

### 棋盘基础渲染
- [x] **BR-01** `BoardRenderer.create(containerId, options)` 创建棋盘实例
- [x] **BR-02** 8×8 棋盘网格渲染（64 个格子，颜色交替正确）
- [x] **BR-03** 棋盘颜色：浅色格 `#eeeed2`，深色格 `#769656`（经典配色）
- [x] **BR-04** 棋盘尺寸自适应容器（支持自定义 `squareSize`）

### 棋子渲染
- [x] **BR-05** MPChess 风格 SVG 棋子（王/后/车/象/马/兵，白/黑各 6 种）
- [x] **BR-06** `BoardRenderer.render(fen?)` 根据 FEN 字符串渲染局面
- [x] **BR-07** 空 FEN 或空参数时渲染空棋盘（无棋子，用于坐标练习）
- [x] **BR-08** 棋子居中显示在格子内，适当缩放

### 坐标标注
- [x] **BR-09** 横坐标标注（a-h）显示在棋盘底部或顶部
- [x] **BR-10** 纵坐标标注（1-8）显示在棋盘左侧或右侧
- [x] **BR-11** 坐标标注颜色：在深色格上用浅色字，在浅色格上用深色字
- [x] **BR-12** 坐标标注字体大小适中，不干扰棋子显示

### 视角反转
- [x] **BR-13** 白方视角：a1 在左下角，h8 在右上角（默认）
- [x] **BR-14** 黑方视角：a1 在右上角，h8 在左下角（棋盘旋转 180°）
- [x] **BR-15** 视角切换时坐标标注随之正确反转
- [x] **BR-16** `options.perspective: 'white'|'black'` 控制视角

### 高亮效果
- [x] **BR-17** `BoardRenderer.highlight(square, type)` 高亮指定格子
- [x] **BR-18** 高亮类型 `correct`：绿色背景高亮
- [x] **BR-19** 高亮类型 `wrong`：红色背景高亮
- [x] **BR-20** 高亮类型 `selected`：蓝色/黄色边框高亮
- [x] **BR-21** `BoardRenderer.clearHighlight()` 清除所有高亮

### 动画效果
- [x] **BR-22** 震动动画：`BoardRenderer.shake()` 用于答错反馈
- [x] **BR-23** 缩放动画：棋子选中时轻微放大
- [x] **BR-24** 脉冲高亮：正确答案闪烁提示
- [x] **BR-25** 动画使用 CSS `@keyframes`，通过添加/移除类名触发
- [x] **BR-26** 动画性能优化：使用 `transform` 和 `opacity`，避免重排

### 交互事件
- [x] **BR-27** `BoardRenderer.onSquareClick(callback)` 注册格子点击回调
- [x] **BR-28** 回调返回点击的格子坐标（如 `'e4'`）
- [x] **BR-29** 点击事件委托在棋盘容器上，避免 64 个单独监听器

### 生命周期
- [x] **BR-30** `BoardRenderer.destroy()` 销毁实例，清理事件监听器
- [x] **BR-31** 防止内存泄漏：移除所有 DOM 引用和事件绑定

### 测试与优化
- [x] **BR-32** 测试 64 格颜色交替正确
- [x] **BR-33** 测试各棋子 SVG 正确显示（12 种）
- [x] **BR-34** 测试白/黑视角下行列反转正确
- [x] **BR-35** 测试高亮和动画类正确添加/移除
- [x] **BR-36** 测试棋盘在移动端显示是否正常（不溢出、可触控）
